# tai-git-integration

## Development

### Backend
```$ gradle build```

```$ java -jar build/libs/git-0.0.1-SNAPSHOT.jar```

Server listen at localhost:8080

### Frontend

```$ npm install```

```$ npm start``` 

Server listen at localhost:4200

### Database 

```$ sudo ./docker_init.sh```

## Build Fat Jar
```$ gradle clean build```

```$ java -jar git-0.0.1-SNAPSHOT.jar```
